# Source Policy

The dashboard prefers official first-party venue, museum, and organization pages.

Third-party listings are an optional fallback tier only for venues where no official parseable calendar exists, is blocked, or can be audited as unavailable.

Rules:
- Official sources always win when they are available and parseable.
- Third-party listings must be labeled clearly in the UI.
- Third-party listings never override official sources.
- Third-party sources can be disabled at any time if they are noisy or stale.
- No social scraping unless it is explicitly approved later.

This document is a plumbing note for future fallback providers such as venue listings on Bandsintown or Songkick.
